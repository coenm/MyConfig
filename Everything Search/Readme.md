## Hotkey

Tools -> Options -> General -> Keyboard
Set "Toggle Window Hotkey" to Ctrl+Shift+Q

## Exclude list

Tools -> Options -> Indexes -> Exclude
 
 - Check 'Enable exclude list'

- C:\Temp
- _ReSharper.Caches
- .svn

- .tmp.drivedownload
- D:\Projects\**\obj
- $Recycle.Bin
- C:\$Recycle.Bin
- D:\$Recycle.Bin
